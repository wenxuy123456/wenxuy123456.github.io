# Amy's Chemistry Experiments

Static website for Amy's chemistry experiments.

## Publish With GitHub Pages

1. Create a public GitHub repository named `YOUR-GITHUB-USERNAME.github.io`.
2. Push this folder to the repository's `main` branch.
3. In GitHub, open `Settings` > `Pages`.
4. Set the source to `Deploy from a branch`, then select `main` and `/root`.
5. Your live site will be available at:

   ```text
   https://YOUR-GITHUB-USERNAME.github.io/
   ```

This version uses GitHub's free `github.io` domain, so no paid domain or DNS setup is needed.
